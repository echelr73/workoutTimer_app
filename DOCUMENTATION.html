🏋️ WorkoutTimer App

Aplicación de entrenamiento offline-first con temporizador configurable, gestión de alumnos y seguimiento de progreso.

Ionic 8 Angular 19 Capacitor 6 SQLite Local PWA + Android Rama: PWA/Release1.1

🏠 Visión General

ℹ️
WorkoutTimer es una app offline-first para entrenadores. Permite definir estructuras HIIT/Tabata, ejecutar un temporizador por fases con audio, registrar alumnos y hacer seguimiento de su progreso — todo almacenado localmente en SQLite.
⏱️ Temporizador
Core de la app
  • Fases: Preparación → Entrenamiento → Descanso → Descanso entre series
  • Countdown 3-2-1 con voces
  • Mantiene pantalla activa (KeepAwake)
  • Play / Pause / Reset
👤 Alumnos & Progreso
Gestión y seguimiento
  • CRUD de alumnos con edad calculada
  • Tabla de ejercicios por alumno
  • Inserción/actualización de medidas
  • Validación de fechas dd/mm/yyyy
🔊 Audio Inteligente
11 beeps + voces por idioma
  • 11 tipos de sonidos de beep
  • Voces en ES e EN por fase
  • Control de volumen 0-10
  • Ducking effect configurable
⚙️ Sin Backend
Arquitectura local
  • Base de datos SQLite local
  • Un solo request HTTP: seed inicial
  • Funciona 100% offline
  • PWA instalable + APK Android

Tab 2 — Estructuras & Timer /tabs/tab2 (ruta por defecto)

💪
Tab principal. Gestiona estructuras de entrenamiento y ejecuta el temporizador multifase. Es la ruta por defecto al abrir la app.
📋 ListStructuresComponent
Lista y gestión de estructuras
  • Lista todas las estructuras de BD
  • Tiempo total formateado (mm:ss)
  • Tap → activa timer con esa estructura
  • Editar → abre FormStructures
  • Eliminar → alerta confirm → DELETE
  • Botón "+" → abre FormStructures (crear)
getStructures() deleteStructure() updateStructure()
📝 FormStructuresComponent
Crear / Editar estructura
  • CustomPickerComponent para min/seg
  • Campos: Nombre, Prep, Entreno, Descanso, Rondas, Series, Descanso entre series
  • Validación Angular + nombre no vacío
  • Calcula tiempo total automáticamente
  • Modo update: @Input() structure
createStructure() updateStructure()
⏱️ TrainingStructureComponent
Motor del temporizador
  • Recibe @Input() structureId
  • Carga estructura desde BD
  • 4 fases con countdown sonoro
  • Estado: isRunning, isPaused, currentPhase
  • KeepAwake activo mientras corre
  • Emite @Output() isRunningTimer
getStructureForId() SoundService.playSound() KeepAwake.keepAwake()

Fórmula de Tiempo Total

totalTime = preparationTime
          + (trainingTime + restTime) × (rounds - 1)
          + trainingTime × series
          + restBetweenSeries × (series - 1)

🔌 API / Endpoints

HTTP Externo

La app realiza un único request HTTP, solo en la primera instalación para sembrar la BD.

GET assets/db/dbTraining.json Seed inicial de la base de datos ▼
PropiedadValor
ServicioSqlliteManagerService.downloadDatabase()
TriggerPrimera vez que se abre la app (no hay BD existente)
Tipo respuestaJsonSQLite (objeto con tablas y datos)
ValidaciónCapacitorSQLite.isJsonValid() antes de importar
Pantallas que lo usanApp init (Tab 2, Tab 3 se benefician del seed)
 
// SqlliteManagerService → downloadDatabase()
this.http.get('assets/db/dbTraining.json')
  .subscribe(async (jsonExport: JsonSQLite) => {
    const jsonString = JSON.stringify(jsonExport);
    const isValid = await CapacitorSQLite.isJsonValid({ jsonstring: jsonString });
    if (isValid.result) {
      await CapacitorSQLite.importFromJson({ jsonstring: jsonString });
    }
  });
✅
Datos iniciales: tabla Configuration con defaults, tabla TrainingStructure con "Tabata" pre-cargado.

Todos los datos persisten localmente. En web usa jeep-sqlite + IndexedDB; en Android usa SQLite nativo.

Servicio de Alertas — AlertService

💬 alertMessage(header, msg)
Alerta informativa
Muestra un diálogo Ionic con un botón "OK". Usado en bienvenida al iniciar la app y mensajes informativos.

Alerta de confirmación
Muestra diálogo con "Cancelar" y "Confirmar". El callback fn se ejecuta al confirmar. Usado en eliminación de estructuras, alumnos y cambios sin guardar.

🌍 Internacionalización

🌐 @ngx-translate
Textos de UI
  • Detecta idioma del dispositivo con Device.getLanguageCode()
  • Usa los primeros 2 caracteres ('es', 'en')
  • Fallback: 'es' por defecto
  • Archivos: assets/i18n/es.json y en.json
  • 60+ claves de traducción
🔊 Voces
Audio localizado
  • SoundService usa el mismo idioma detectado
  • Mapea voiceMap[idioma][fase] a ruta de archivo
  • Reproduce con HTMLAudioElement nativo
  • Si el idioma no está soportado → usa 'en'
